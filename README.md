# Finance Tracker - UI (React & Material UI) 

This is the frontend application of the **Finance Tracker** project, designed to provide a professional and intuitive user experience for personal finance management. It is built with **React** and **Material UI (MUI)**, communicating with a robust **.NET 8 Web API** backend.

##  Overview
Managing expenses and tracking income is simplified through this modern web interface. This project was developed to demonstrate full-stack capabilities, focusing on clean UI, secure authentication, and real-time financial data visualization, drawing from my 2+ years of experience in the **Fintech** sector.

## Key Features
* **Professional UI with MUI:** Developed using **Material UI** components for an enterprise-grade, responsive design.
* **User Authentication:** Secure login and registration system using JWT, managed via **AuthContext**.
* **Dynamic Dashboard:** A comprehensive overview of financial status with visual charts and analytics.
* **Transaction Management:** Effortlessly add, edit, or delete income and expense records.
* **Category & Admin Operations:** Organize finances by custom categories and access admin-specific panels.
* **Responsive Layout:** Optimized for both desktop and mobile viewing using MUI's flexible grid system.
